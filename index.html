How to use

1. Watch the chartTrack monthly changes in house or unit trend.

Demo data is preloaded so the file works offline except for the Chart.js library. Replace the dataset in the script block with your own monthly figures.

Cycle viewMelbourne has been slower than several other capitals recently, which can create better entry points if rents stay resilient.
Use togetherTrack both broad price indices and suburb-level medians. One tells you the cycle; the other tells you where value sits.
